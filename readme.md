# My git commands

## git basic commands

  stage a file

```git
  git add <file>
```

  remove a file from git directory

```git
  git rm --cached <file>
```

  make a commit

```git
  git commit
```

  make a commit without delete other message

```git
  git commit --amend --no-edit
```

  to commit with a message
```git
  git commit --message
```
